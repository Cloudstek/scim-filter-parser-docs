Disjunction

Example

foo eq "bar" or baz ew "S" — translates to a Disjunction node with two Comparison nodes.

Implements

  • Node - Base interface for all AST nodes.
  • Connective - Interface for logical operations
    • Countable - Count the number of nodes
    • Array access - Access the nodes as indexed array
    • Iterator - Iterate over the nodes directly

Methods

getNodes(): Node[]

Returns all nodes.
